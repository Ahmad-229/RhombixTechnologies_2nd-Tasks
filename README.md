# Containerization with Docker - Project 2

## Project Description
This project demonstrates containerization of an application using Docker. It includes building a Docker image, running a container, and exposing the application.

## Steps Covered
- Created Dockerfile
- Built Docker image
- Ran container locally
- Tested application in browser/terminal
- Pushed project to GitHub

## How to Run

docker build -t my-app .
docker run -p 8080:8080 my-app

- Docker
